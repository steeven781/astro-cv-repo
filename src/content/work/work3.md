- Engineered targeted test cases from requirements and user stories, tailoring scenarios to diverse testing conditions to maximize defect detection and system coverage.

- Developed structured test plans for Functional, Regression, Exploratory, Smoke, and E2E testing, achieving full test coverage across the entire product lifecycle.
- Systematically tracked, documented, and resolved defects across APIs, web applications, and databases, contributing to stable and reliable production releases.
- Automated API validation by authoring and executing test steps, plans, and scripts with Postman collections, improving consistency and repeatability of API quality checks.
- Kept automated test plans and Postman collections continuously updated, ensuring alignment with evolving product requirements and preventing test drift.
- Integrated automation scripts into GitLab CI/CD pipelines, enabling quality gates within the continuous integration workflow and accelerating delivery confidence.
- Upheld the accuracy and completeness of a living Test Plan, maintaining focused functional coverage for both existing features and incoming development.
- Utilized Jira for holistic project tracking — managing user stories, bugs, tasks, and work hours — ensuring full sprint visibility and efficient project delivery.
